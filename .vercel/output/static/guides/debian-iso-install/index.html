Installing Debian ISO on USB stick

I am not running a hypervisor such as Proxmox since I want to run kubernetes bare-metal in Debian. So I’ll need to put the Debian ISO on a USB stick so that I can install it each Mini PC. I went with Debian as my Linux OS since it has excellent documentation and is relatively stable.

Setup Instructions on a Mac

Danger

These instructions are for installing on a Mac only, and make sure you have a USB stick with at least 8 GB of free memory.

  1. Plug the USB stick in.

  2. Go to Debian distribution page and click on amd64 since that’s the architecture for our N100 Intel chip to download the ISO file.

  3. Cd into the downloads folder where the ISO file is.

  4. List all of the disks on the mac. You should see one listed for (external, physical). That’s the USB stick, make note of the disk name. For me it is /dev/disk4.

  5. Now we are going to erase the contents of the disk completely. Replace /dev/disk4 with your disk name for this command and all following commands.

  6. Since erasing the content mounts the usb disk, we need to unmount it before adding our ISO file.

  7. Finally, let’s add the ISO file to the disk. Change the file name to be the name of your iso file along with the disk name.

    Note

    You will notice an error after this step saying the disk is not readable, click eject anyways, this error is expected since Macs can’t read the new filesystem on the USB stick.
